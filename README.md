- Precipitation trends across regions and seasons
- KPIs: monthly/annual rainfall, anomalies
- Climate trend analysis and resource planning

- Lifestyle indicators and public health metrics
- KPIs: obesity rate, smoking prevalence, inactivity levels
- Demographic breakdowns for awareness and reporting
